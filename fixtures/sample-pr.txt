## Author

## Body
This PR adds a new CLI `preview` that turns URLs, markdown, JSON, and command output into interactive terminal UIs.

### Features
- Docs mode for HTML pages
- API/JSON mode with table and detail view
- Markdown mode with TOC
- GitHub PR mode for `gh pr view` output
